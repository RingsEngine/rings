glTF 骨骼动画

使用 CDN 上的 eva_unit_01_rigged.glb（resources-0.0.2）。loadGltf 会为带蒙皮的节点挂上 AnimatorComponent，并将 glTF 动画解析为 PropertyAnimationClip； 蒙皮网格由 SkinnedMeshRenderer2 读取骨骼矩阵。程序化 SkeletonAnimationClip + SkeletonAnimationComponent 的写法见本仓库其它示例（手写 buffer 管线）。

若 glTF 单位很小，示例会对根节点放大并对 BoundUtil.genMeshBounds 结果做 focusByBounds， 避免相机仍在原点附近、视距过近而「在模型内部」或模型只占亚像素。
