肩 / 肘 / 腕 + 五指 均为 HingeConstraint；底部播放用 enableAngularMotor 做拟人摆动；左键拖拽可手动摆姿态。
加载约束后可点按；肩/肘/腕相位错开，模拟挥手与屈伸。

操作

拖拽使用 Physics.init({ useDrag: true })（与核心、物理均从 unpkg 加载）。

播放：对肩、肘、腕铰链施加正弦式目标角速度（enableAngularMotor），再点一次停止。

左键空白 旋转视角；右键平移；滚轮缩放。

核心与物理均自 unpkg 加载。请用 HTTP 服务打开（localhost / HTTPS），勿用 file://。
