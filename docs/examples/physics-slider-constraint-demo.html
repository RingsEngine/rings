SliderConstraint 属性（摘要）

对应 btSliderConstraint；下列除「继承」外均在滑块物体上 addComponent 后赋值。

  • 继承 ConstraintBase：targetRigidbody（导轨刚体）、pivotSelf / pivotTarget（两侧局部锚点）、rotationSelf / rotationTarget（约束框朝向，未设则为单位四元数）、disableCollisionsBetweenLinkedBodies、breakingThreshold
  • useLinearReferenceFrame：是否使用线性参考系（默认 true，传入 Ammo 构造）
  • lowerLinLimit / upperLinLimit：沿滑动轴的线位移下限 / 上限（默认近似无界）
  • lowerAngLimit / upperAngLimit：绕约束相关轴的角度限制（弧度；本页设为 0 抑制扭转）
  • poweredLinMotor：是否启用线性马达
  • maxLinMotorForce：马达最大推力
  • targetLinMotorVelocity：马达目标线速度（沿滑动轴）
