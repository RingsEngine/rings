# Class: GlobalIlluminationComponent

动态全局光照.
使用全局光照实现更真实的光照效果。
全局光照系统可以模拟光线在表面反射或折射到其他表面(间接光照)的方式，
而不仅限于光线直接从光源照射到某个表面。
