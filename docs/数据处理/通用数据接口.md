## `ParserFormat`

| 枚举 | `FileLoader` 行为 |
|------|-------------------|
| `BIN` | `loadBinData` → `parseBuffer` |
| `JSON` | `loadJson` → `parseJson` |
| `TEXT` | `loadTxt` → `parseString` |

每个 Parser 子类声明 `static format`，例如：

```typescript
class MyParser extends ParserBase {
  static format = ParserFormat.JSON;
  async parseJson(obj: object) { this.data = obj; }
  verification() { return !!this.data; }
}
```

## `ParserBase`

| 成员 | 说明 |
|------|------|
| `baseUrl` | 相对资源根路径 |
| `initUrl` | 初始请求 URL |
| `loaderFunctions` | 加载回调 |
| `userData` | 调用方自定义数据 |
| `data` | 解析结果 |
| `parseBuffer` / `parseJson` / `parseString` | 子类实现其一 |
| `verification()` | 成功返回 `true`，失败应抛错 |

## `FileLoader`

```typescript
load<T extends ParserBase>(
  url: string,
  c: Parser<T>,
  loaderFunctions?: LoaderFunctions,
  userData?: any
): Promise<T>
```

流程：按 `c.format` 取字节/JSON/文本 → `new c()` → 赋值上下文 → 调用对应 `parse*` → `verification()`。

## `LoaderFunctions`

| 字段 | 说明 |
|------|------|
| `onProgress` | 进度 |
| `onComplete` | 完成 |
| `onError` | 错误 |
| `onUrl` | URL 重写 |
| `headers` | 请求头 |

## `Res.load` 泛型

```typescript
const data = await Engine3D.res.load(url, GeoJsonParser);
// 等价于 (await new FileLoader().load(url, GeoJsonParser)).data
```

`Res.loadJSON` 则不经 Parser，直接返回 JSON 对象。

## 扩展新格式

1. 继承 `ParserBase`，设置 `static format`。
2. 实现对应 `parse*` 与 `verification()`。
3. 在 `src/index.ts` 导出（若需公开 API）。
4. 业务侧：`new FileLoader().load(url, MyParser)` 或 `Engine3D.res.load(url, MyParser)`。
