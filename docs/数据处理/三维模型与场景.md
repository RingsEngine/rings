# 三维模型与场景

加载 glTF / GLB / OBJ 及 Rings 预制（Prefab）为可挂载的 `Object3D` 场景树。

## 可运行示例

需通过 **HTTP(S)** 打开文档站，浏览器需支持 **WebGPU**。

## Res 封装 API

| 方法 | 参数 | 返回值 | 说明 |
|------|------|--------|------|
| `loadGltf(url, loaderFunctions?)` | URL 以 `.gltf` 或 `.glb` 结尾 | `Promise<Object3D>` | 内部选择 `GLTFParser` / `GLBParser`，并缓存至 `_prefabPool`、`_gltfPool` |
| `loadObj(url, loaderFunctions?)` | `.obj` | `Promise<Object3D>` | 使用 `OBJParser` |
| `load(url, ParserClass, loaderFunctions?)` | 任意 Parser | `Promise<parser.data>` | 泛型加载 |
| `getGltf(url)` | 已加载 URL | `GLTF_Info` | 访问 glTF 元数据（材质、动画等） |

## 解析器类

### `GLTFParser` / `GLBParser`

| 成员 | 类型 | 说明 |
|------|------|------|
| `static format` | `ParserFormat.BIN` / `JSON` | GLB 为 BIN；glTF 主文件为 JSON |
| `data` | `Object3D` | 解析后的场景根节点 |
| `gltf` | `GLTF_Info` | 原始 glTF 结构（仅 GLB/GLTF 路径） |
| `parseBuffer` / `parseJson` | — | 由 `FileLoader` 调用 |

### `OBJParser`

| 成员 | 说明 |
|------|------|
| `static format` | `ParserFormat.TEXT` |
| `data` | `Object3D` 根节点 |

### `PrefabParser`

| 成员 | 说明 |
|------|------|
| `static format` | `ParserFormat.BIN` |
| `static useWebp` | 纹理是否按 WebP 解析 |
| `data` | `Object3D` |
| `avatarDic` / `nodeData` | 预制内部结构 |

子模块：`PrefabMeshParser`、`PrefabMaterialParser`、`PrefabTextureParser`、`PrefabAvatarParser`。

## 使用示例

```typescript
import { Engine3D, FileLoader, GLBParser, PrefabParser } from "@rings-webgpu/core";

// 推荐：Res 缓存
const duck = await Engine3D.res.loadGltf("https://example.com/Duck.glb");
scene.addChild(duck);

const meta = Engine3D.res.getGltf("https://example.com/Duck.glb");

// 直接 Parser
const loader = new FileLoader();
loader.baseUrl = "https://example.com/assets/";
const parser = await loader.load("model.glb", GLBParser);
scene.addChild(parser.data);
```
