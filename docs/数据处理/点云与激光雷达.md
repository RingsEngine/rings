# 点云与激光雷达

支持 **PLY 点云**（`PlyParser` / `PlyMode.PointCloud`）与 **LAS / LAZ**（`LASParser`，含测井曲线与井轨迹模式）。

## 可运行示例

示例从 CDN 加载 `biker.ply`（`PlyParser` 按文件头自动识别点云 / 3DGS 等模式）。

## `PlyParser`（点云模式）

| 成员 | 说明 |
|------|------|
| `static format` | `ParserFormat.BIN` |
| `static cloudImageProcessParam` | 网格 PLY 纹理 URL 后缀（点云不用） |
| `data` | `Object3D`，子节点含 `PointCloudRenderer` |

识别规则：PLY 头含 `x,y,z` 与颜色属性，且无 face 元素 → `PlyMode.PointCloud`。

## `PointCloudRenderer`（组件）

| 方法 / 属性 | 说明 |
|-------------|------|
| `initFromData(positions, colors, count)` | `Float32Array` 位置 + `Uint8Array` RGBA |
| `setPointSize(size)` | 点大小 |
| `setPointShape(shape)` | 如 `'circle'` |
| `centerOffset` | 中心偏移 |

## `LASParser`

| 成员 | 说明 |
|------|------|
| `static format` | `ParserFormat.BIN` |
| `visualizationMode` | `LASVisualizationMode`：`PointCloud` / `Curve` / `WellTrajectory` |
| `data` | `Object3D` 可视化根 |

`LASLoader` 自动区分 ASCII 测井 LAS 与二进制 LIDAR（魔数 `LASF`）。

### `LASVisualizationMode`

| 枚举值 | 用途 |
|--------|------|
| `PointCloud` | 激光点云 |
| `Curve` | 测井曲线 |
| `WellTrajectory` | 井轨迹 |

## 使用示例

```typescript
import { FileLoader, PlyParser, LASParser, LASVisualizationMode } from "@rings-webgpu/core";

// PLY 点云
const plyRoot = await Engine3D.res.load("scan.ply", PlyParser);
scene.addChild(plyRoot);

// LAS（二进制或 ASCII）
const loader = new FileLoader();
const las = await loader.load("well.las", LASParser);
las.visualizationMode = LASVisualizationMode.WellTrajectory;
await las.parseBuffer(buffer); // 通常由 FileLoader 完成
scene.addChild(las.data);
```
