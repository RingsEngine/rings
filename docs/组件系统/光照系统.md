# 光照系统

### 光源类型

#### 平行光 (DirectLight)

模拟太阳光等平行光源，光线方向一致，无限范围。

**属性说明**

| 属性名 | 类型 | 默认值 | 描述 |
|--------|------|--------|------|
| `lightColor` | Color | 白色 | 光源颜色（RGBA） |
| `intensity` | number | 1.0 | 光照强度 |
| `castShadow` | boolean | false | 是否投射阴影 |
| `castGI` | boolean | false | 是否参与全局光照 |
| `indirect` | number | 0.3 | 间接光强度 |
| `radius` | number | Number.MAX_SAFE_INTEGER | 光照范围（无限远） |
| `size` | number | 1 | 光源大小 |
| `realTimeShadow` | boolean | true | 是否实时更新阴影 |
| `needUpdateShadow` | boolean | true | 是否需要更新阴影 |
| `iesProfile` | IESProfiles | null | IES光度学文件 |
| `shadowCamera` | Camera3D | null | 阴影相机实例 |

**使用示例**

```typescript
import { Object3D, Scene3D, DirectLight, Color } from '@rings/core';

// 创建场景
const scene = new Scene3D();

// 创建平行光源
const lightObj = new Object3D();
const light = lightObj.addComponent(DirectLight);

// 配置光源属性（方向由物体旋转决定 forward）
light.lightColor = new Color(1, 1, 1, 1);
light.intensity = 1.0;
light.castShadow = true;
light.castGI = true;
light.indirect = 0.5;

// 设置光照方向
lightObj.transform.rotationX = -45;
lightObj.transform.rotationY = 45;

// 添加到场景
scene.addChild(lightObj);
```

#### 点光源 (PointLight)

模拟灯泡等点状光源，光线向四周发散。

**属性说明**

| 属性名 | 类型 | 默认值 | 描述 |
|--------|------|--------|------|
| `lightColor` | Color | 白色 | 光源颜色（RGBA） |
| `intensity` | number | 1.0 | 光照强度 |
| `range` | number | 10 | 光照范围 |
| `castShadow` | boolean | false | 是否投射阴影 |
| `castGI` | boolean | false | 是否参与全局光照 |
| `radius` | number | 0 | 光源半径 |
| `size` | number | 1 | 光源大小 |
| `at` | number | 0 | 线性衰减系数 |
| `quadratic` | number | 1 | 二次衰减系数 |
| `realTimeShadow` | boolean | true | 是否实时更新阴影 |
| `needUpdateShadow` | boolean | true | 是否需要更新阴影 |
| `iesProfile` | IESProfiles | null | IES光度学文件 |

**使用示例**

```typescript
import { Object3D, Scene3D, PointLight, Color, Vector3 } from '@rings/core';

// 创建场景
const scene = new Scene3D();

// 创建点光源
const lightObj = new Object3D();
const light = lightObj.addComponent(PointLight);

// 配置光源属性
light.lightColor = new Color(1, 0.5, 0, 1);
light.intensity = 2.0;
light.range = 15;
light.radius = 1;
light.at = 0.1;
light.quadratic = 0.5;
light.castShadow = true;

// 设置光源位置
lightObj.transform.position = new Vector3(5, 5, 0);

// 添加到场景
scene.addChild(lightObj);
```

#### 聚光灯 (SpotLight)

模拟手电筒等聚光效果，具有方向性和锥形范围。

**属性说明**

| 属性名 | 类型 | 默认值 | 描述 |
|--------|------|--------|------|
| `lightColor` | Color | 白色 | 光源颜色（RGBA） |
| `intensity` | number | 1.0 | 光照强度 |
| `range` | number | 10 | 光照范围 |
| `angle` | number | 30 | 外锥角（度） |
| `penumbra` | number | 0.2 | 边缘衰减 |
| `castShadow` | boolean | false | 是否投射阴影 |
| `castGI` | boolean | false | 是否参与全局光照 |
| `innerAngle` | number | 100 | 内锥角百分比（0-100） |
| `outerAngle` | number | 60 | 外锥角（度，1-179） |
| `radius` | number | 0 | 光源半径 |
| `size` | number | 1 | 光源大小 |
| `at` | number | 0 | 线性衰减系数 |
| `quadratic` | number | 1 | 二次衰减系数 |
| `realTimeShadow` | boolean | true | 是否实时更新阴影 |
| `needUpdateShadow` | boolean | true | 是否需要更新阴影 |
| `iesProfile` | IESProfiles | null | IES光度学文件 |

**使用示例**

```typescript
import { Object3D, Scene3D, SpotLight, Color, Vector3 } from '@rings/core';

// 创建场景
const scene = new Scene3D();

// 创建聚光灯
const lightObj = new Object3D();
const light = lightObj.addComponent(SpotLight);

// 配置光源属性
light.lightColor = new Color(0.8, 0.8, 1, 1);
light.intensity = 3.0;
light.range = 20;
light.outerAngle = 45;
light.innerAngle = 80; // 内锥相对外锥的百分比（0-100）
light.at = 0.1;
light.quadratic = 0.5;
light.castShadow = true;

// 设置光源位置和方向（lookAt(光源位置, 目标点)）
const pos = new Vector3(0, 10, 0);
const target = new Vector3(0, 0, 0);
lightObj.transform.position = pos;
lightObj.transform.lookAt(pos, target);

// 添加到场景
scene.addChild(lightObj);
```

### 环境光照

#### 环境光 (AmbientLight)

提供场景基础环境照明，无方向性。

> **说明**：当前 `@rings/core` / `@rings-webgpu/core` **未导出** `AmbientLight` 组件。实践中常用 **AtmosphericComponent（天空/大气）** 配合 **大范围、低强度的 PointLight**（或多盏弱光）模拟均匀环境搭光；也可通过 **DirectLight** 的 **`indirect`** 增加间接感。

**属性说明（规划 / 其它引擎常见命名，供对照）**

| 属性名 | 类型 | 默认值 | 描述 |
|--------|------|--------|------|
| `color` / `lightColor` | Color | 灰色 | 环境光颜色 |
| `intensity` | number | 0.2 | 光照强度 |

**使用示例（推荐：大气 + 弱填充点光）**

```typescript
import { Object3D, Scene3D, AtmosphericComponent, PointLight, Color, Vector3 } from '@rings/core';

const scene = new Scene3D();
scene.addComponent(AtmosphericComponent);

// 两盏弱点点光，大范围，模拟无方向环境填充
const fillA = new Object3D();
const plA = fillA.addComponent(PointLight);
plA.lightColor = new Color(0.85, 0.9, 1, 1);
plA.intensity = 0.12;
plA.range = 40;
plA.castShadow = false;
fillA.transform.position = new Vector3(0, 6, 0);
scene.addChild(fillA);

const fillB = new Object3D();
const plB = fillB.addComponent(PointLight);
plB.lightColor = new Color(1, 0.95, 0.88, 1);
plB.intensity = 0.1;
plB.range = 35;
plB.castShadow = false;
fillB.transform.position = new Vector3(-6, 3, 4);
scene.addChild(fillB);
```
