# 网格 (Mesh)

网格是3D场景中的核心渲染对象，由几何体(Geometry)和材质(Material)组成，支持多种渲染特性和优化选项。

### 网格系统架构

| 网格类型 | 类名 | 继承关系 | 主要特性 |
|----------|------|----------|----------|
| **基础网格** | `MeshRenderer` | `RenderNode` → `MeshRenderer` | 标准网格渲染 |
| **网格过滤器** | `MeshFilter` | `MeshRenderer` → `MeshFilter` | 几何体管理 |
| **蒙皮网格** | `SkinnedMeshRenderer` | `MeshRenderer` → `SkinnedMeshRenderer` | 骨骼动画支持 |
| **蒙皮网格2** | `SkinnedMeshRenderer2` | `MeshRenderer` → `SkinnedMeshRenderer2` | 优化版蒙皮渲染 |

### 各类网格使用示例

四类组件均继承 **`MeshRenderer`**，**`geometry` / `material` / `materials`**、**`castShadow` / `receiveShadow`**、**`renderLayer`** 等通用用法一致；差异在**几何体来源**（自建、资源池 URL、带蒙皮数据）以及**骨骼组件**（`SkeletonAnimationComponent` / `AnimatorComponent`）。

#### `MeshFilter`（按资源键取几何体）

通过 **`meshURL`** 从 **`Engine3D.res`** 中**已注册**的几何体读取（内部为 **`getGeometry(url)`**）。需先用 **`Engine3D.res.addGeometry(url, geo)`** 放入池子。**推荐**使用 `meshURL`；也可与 `MeshRenderer` 一样直接赋 `geometry`。

#### 可运行示例（MeshFilter + Engine3D.res.addGeometry）

先用 **`Engine3D.res.addGeometry(key, geo)`** 注册，再用 **`MeshRenderer.geometry = res.getGeometry(key)`**（与 **`MeshFilter.meshURL`** 等价，避免旧版 CDN 上 MeshFilter 未写入几何体导致白屏）。

#### `SkinnedMeshRenderer`（蒙皮 · `SkeletonAnimationComponent`）

需配置 **`skinJointsName`**、**`skinInverseBindMatrices`**，并与 **`SkeletonAnimationComponent`**（同节点或父级可解析到）配合。新资源与 GLTF 流程更推荐下面的 **`SkinnedMeshRenderer2`**。下例为同一 RiggedSimple 场景：GLTF 实际挂载 **`SkinnedMeshRenderer2`**，HUD 会统计 **`SkinnedMeshRenderer`（旧）** 数量（通常为 0）。

### 更多用法示例

#### 多材质网格

几何体含**多子网格 / 多材质槽**时，使用 **`materials`** 数组（顺序与几何体子网格对应）。

```typescript
import { Object3D, MeshRenderer, LitMaterial, Color } from '@rings-webgpu/core';

const obj = new Object3D();
const mr = obj.addComponent(MeshRenderer);
// mr.geometry = geoFromLoader; // 多子网格几何体

const wall = new LitMaterial();
wall.baseColor = new Color(0.55, 0.35, 0.22, 1);
wall.roughness = 0.8;
const roof = new LitMaterial();
roof.baseColor = new Color(0.4, 0.4, 0.42, 1);
roof.roughness = 0.35;
const glass = new LitMaterial();
glass.baseColor = new Color(0.25, 0.45, 0.85, 1);
glass.roughness = 0.15;

mr.materials = [wall, roof, glass];
scene.addChild(obj);
```

#### GPU 实例化（`instanceCount`）

**`instanceCount > 0`** 时走实例化绘制；每实例矩阵等数据需按项目管线写入 **Uniform / Storage**（以下为占位示意）。

```typescript
import { Object3D, MeshRenderer, BoxGeometry, LitMaterial, Color, Matrix4 } from '@rings-webgpu/core';

const obj = new Object3D();
const mr = obj.addComponent(MeshRenderer);
mr.geometry = new BoxGeometry(0.5, 0.5, 0.5);
mr.material = new LitMaterial();
mr.material.baseColor = new Color(0.2, 0.9, 0.4, 1);
mr.instanceCount = 1000;

const instanceMatrices: Matrix4[] = [];
for (let i = 0; i < 1000; i++) {
  const m = new Matrix4();
  m.setPosition(Math.random() * 100 - 50, Math.random() * 10, Math.random() * 100 - 50);
  instanceMatrices.push(m);
}
// 将 instanceMatrices 提交到引擎/材质侧实例缓冲（依版本 API 而定）

scene.addChild(obj);
```

#### LOD（`lodLevel`）

为不同距离准备多套 **`MeshRenderer`**（或子物体），分别设置 **`lodLevel`**，由场景/LOD 策略切换显示（具体切换逻辑依项目）。

```typescript
import { Object3D, MeshRenderer, BoxGeometry, LitMaterial } from '@rings-webgpu/core';

const lodRoot = new Object3D();

const high = new Object3D();
const r0 = high.addComponent(MeshRenderer);
r0.geometry = new BoxGeometry(2, 2, 2);
r0.material = new LitMaterial();
r0.lodLevel = 0;
lodRoot.addChild(high);

const low = new Object3D();
const r1 = low.addComponent(MeshRenderer);
r1.geometry = new BoxGeometry(2, 2, 2);
r1.material = new LitMaterial();
r1.lodLevel = 1;
lodRoot.addChild(low);

scene.addChild(lodRoot);
```

#### 动态批处理

相同 **`geometry` + `material`** 的多份 **`Object3D`** 更易被合批；可用 **`clone()`** 复制节点。

```typescript
import { Object3D, MeshRenderer, BoxGeometry, LitMaterial, Vector3 } from '@rings-webgpu/core';

const sharedLitMaterial = new LitMaterial();

const proto = new Object3D();
const meshRenderer = proto.addComponent(MeshRenderer);
meshRenderer.geometry = new BoxGeometry(1, 1, 1);
meshRenderer.material = sharedLitMaterial;

for (let i = 0; i < 100; i++) {
  const inst = proto.clone();
  inst.localPosition = new Vector3(i * 2, 0, 0);
  scene.addChild(inst);
}
```
